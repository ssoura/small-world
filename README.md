# Small World

> React Web App that helps you find the degree of separation between any two people.
> Think of it as selecting two users on Facebook and trying to see how these two people are
> connected.

## Install

#### pnpm

```sh
pnpm install
pnpm run dev
```

#### npm

```sh
npm install
npm run dev
```
